n   添加批次

1.       保证已登录系统。若未登录，请登录系统，例以用户名为admin，密码为admin的身份登录

图1 登录系统

2.       在主界面选择左侧导航栏中的“新建批次”或点击右侧的“新建批次”，均可即进入添加批次界面

 

 

 

 

图2 进入新建批次主界面

3.       输入批次信息。

在添加界面输入产品编号，自动显示该批次的批次号和对应的生产流程。这里注意：填入某个输入框在另一个输入框自动生成信息，这种形式均是在前者输入框失去焦点时产生的，即第一个输入框输入完毕，点击除该输入框其他地方时，另一个输入框可显示相应信息。以下自动生成信息的操作也是如此，不再赘述。

其中批次号的组成结构为：日期（YYYYMMDD）+递增序号（NN），如2012082501；生产流程显示该产品的所有可能的生产流程且默认选中当前使用的最新的流程。

填写该批次的目标数量，则该批次的实际生产数量为：目标数量+递增序号，如再填入目标数量为1000，则实际数量为1001。当目标数量为空时，系统默认目标数量为0。

若输入产品编号有误或者产品未导入数据库，则流程一栏显示“未找到相应流程！请重新输入产品编号”，批次号一栏显示“undefined”，此时，点击提交，会弹出提示框“undefined”，需要检查产品编号是否正确或该产品是否已导入。如同时生产的批次已有20个，此时批次号一栏显示“该产品批次已满！”，此时，点击提交，会弹出提示框“该产品批次已满！”，需要检查产品编号是否正确或该产品是否已导入。

提交成功后，原界面显示“添加成功”，同时自动打开生产的pdf页面。否则，原界面显示“添加失败”或其失败原因，此时请重复以上添加过程，重新操作。

如以JL466Q气门导管为例。

n   添加工单

1. 保证已登录系统。若未登录，请登录系统，例以用户名为admin，密码为admin的身份登录

图7 登录系统

2.       在主界面选择左侧导航栏中的“添加工单”或点击右侧的“添加工单”，均可即进入添加工单界面

图8 进入添加工单主界面

3.       扫描条形码

扫描成功后自动进入添加工单输入界面。否则会提示进行扫描。

图9 扫描条形码，相当于键入扫描条码后点击回车

图10 进入工单输入界面

4.       输入工单内容，进行添加

在输入界面，输入工序编号，显示工序名称。输入员工工号，显示员工姓名。当合格品或不合格品数量输入为空时，默认该项数量为0

若输入工序编号有误，则输入该编号后会立即提示“工序或批次有误”。

对于该批次的第一张工单，当该工单中所有合格品数量+不合格品数量不等于该批次的目标数量时，显示“后工序大于前工序数量”的错误提示。除此外，当次工单代表的工序所有的合格品数+不合格品数量不等于上一道工序的合格品数量时，同样会显示“后工序大于前工序数量”的错误提示。此时需要检查该工单的产品数量是否正确或前序所有工单是否有错误工单。

图11工单输入。红色为需要输入部分

5. 添加工单成功后进入工单列表界面，且可选择“继续添加工单”进行添加。

图12工单列表。

6． 在工单列表界面，可选择列表记录中后面的图标，查看该工单的详细内容。

图13 查看工单详情。

n   添加返工工单

1.       确保已登录系统，若未登录，则按照前面介绍进行登录。

2.       在主界面选择左侧导航栏中的“添加返工工单”或点击右侧的“添加返工工单”，均可即进入添加返工工单界面

图14 进入添加返工工单界面。

图15 返工工单输入界面。

3.       在输入界面，输入产品编号，显示产品名称。输入工序号，显示工序名称。输入员工工号，显示员工姓名。输入责任人工号和审批人工号，显示两人的姓名。

同时，若输入的编号不存在，则会提示响应的不存在信息。比如，若输入不存在的员工工号，则提示，该员工不存在。此时要检查输入编号是否正确，或检查该编号对应的员工或产品或工序是否在数据库内存在。

如以下图为例

图16 返工工单进行输入。

4. 原界面显示添加结果。以下图为例。

图17 返工工单输入结果。

5.  显示返工工单列表。

在主界面选择左侧导航栏中的“返工工单列表”或点击右侧的“返工工单列表”，均可即进入返工工单列表搜索界面。在搜索界面选择搜索项，如“按完成员工搜索”，并输入搜索条件，如工号为11011，可搜索出该工人所完成的所有返工工单。

同时，若搜索条件所对应的工号不存在，则会在搜索界面显示不存在信息。

在返工工单列表界面，点击，可显示该返工工单的详细信息。
